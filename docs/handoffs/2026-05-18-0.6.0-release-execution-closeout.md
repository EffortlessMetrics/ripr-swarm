## Public Copy Boundary

`cargo xtask check-product-copy` passed after publish. Public release surfaces
preserve the release boundary:

- ripr is static mutation-exposure guidance.
- Rust is the stable evidence path.
- TypeScript/JavaScript/Python are preview/advisory.
- Mutation testing remains the runtime backstop.
- ripr does not generate tests, edit source, call providers, or prove
  coverage/correctness.
- CI gates remain explicit policy choices, not default authority.
