# ripr SVG assets

These are initial draft vector starting points, and may still evolve.

## Recommended usage

- `ripr-icon-dark.svg`: VS Code Marketplace, Open VSX, GitHub avatar, favicon base.
- `ripr-icon-dark.png`: generated marketplace/package icon used as `editors/vscode/icon.png`.
- `ripr-icon-light.svg`: light-mode icon variant.
- `ripr-rail-transparent.svg`: inline rail mark; inherits `currentColor` for the teal rail and uses fixed amber for the reveal node.
- `ripr-lockup-dark.svg`: docs/social/header lockup; convert text to outlines once final.
- `ripr-lockup-light.svg`: light-mode lockup; convert text to outlines once final.
- `ripr-wordmark-dark.svg`: wordmark-only presentation; convert text to outlines once final.

## Colors

- Dark background: `#1A1816`
- Warm paper: `#F5F1E8`
- Dark-mode teal: `#1D7D8A`
- Light-mode teal: `#156B76`
- Dark-mode amber: `#F59E0B`
- Light-mode amber: `#E1A82C`
- Dark-mode wordmark: `#F0ECE5`
- Light-mode wordmark: `#1A1A1A`
