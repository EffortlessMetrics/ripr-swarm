# Allowlisted dependency-management surfaces.
#
# Format:
# glob|kind|owner|reason
#
# New dependency manager files or manifests require an allowlist entry with an
# owner and reason. Keep dependency surfaces scoped to Cargo, the VS Code
# extension, and fixture input workspaces.
#
# Approved crate decisions:
# ra_ap_syntax|production_analyzer|analysis|ADR 0006 approves Rust-native
# parser-backed syntax extraction; add the Cargo dependency only in the scoped
# parser PR.
# oxc_parser|production_analyzer|analysis|ADR 0008 approves Rust-native
# TypeScript/JavaScript syntax extraction for the Campaign 27 preview adapter;
# add the Cargo dependency only in the scoped parser PR.
# rustpython-parser|production_analyzer|analysis|ADR 0009 approves Rust-native
# Python syntax extraction for the Campaign 27 preview adapter; add the Cargo
# dependency only in the scoped parser PR. The ADR's original `ruff_python_parser`
# pick was superseded after discovering that crate is `publish = false` inside
# the astral-sh/ruff workspace and unavailable on crates.io.
